# Supply Chain Analytics and Inventory Optimization System

## Overview
This project performs supply chain analytics using Python and a real-world dataset. It analyzes inventory, suppliers, transportation costs, and product performance to generate insights for supply chain optimization.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Features

### KPI Analysis
- Total Revenue
- Average Stock Level
- Average Defect Rate
- Total Inventory Value

### Inventory Management
- Low Stock Detection
- Stock Status Classification
- Reorder Point Calculation
- Reorder Alerts
- Safety Stock

### Product Analysis
- ABC Classification

### Supplier Analysis
- Revenue by Supplier
- Defect Rate Analysis
- Best Supplier Identification

### Transportation Analysis
- Cost Comparison of Air, Rail, Road and Sea

### Visualizations
- Revenue by Product Type
- Supplier Revenue
- Transportation Cost Analysis
- Supplier Quality Analysis
- Inventory Value Distribution
- ABC Classification
